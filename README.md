Это сайт, сделанный на React + SQL DB
Инструкция по запуску проекта:

#### `cd api /cd client`
```bash
npm i 
```
При помощи Xampp запустить MYSQL 
В MYSQL загрузить БД из папки "MYSQL"
В папке api/db.js Ввести данные БД

#### `cd api`
```bash
npm run start```

#### `cd client`
```bash
npm run start```
